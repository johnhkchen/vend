## Context

`mdwc` counts the words in a single Markdown file and prints the integer. This
story delivered the entire product (see the charter — scope is complete and
frozen). It is **done**: the tool reads a file path, strips markup, counts words,
prints the count, exits `0`. Nothing about it remains open.

## Acceptance Criteria

- [x] Reads one Markdown file path from `argv` and prints the integer word count.
- [x] Empty or absent file prints `0`, never an error.
- [x] No flags, no config, no dependencies.
