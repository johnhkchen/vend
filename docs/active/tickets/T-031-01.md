## Context

The pure heart of the DL-6 Home: a one-line ledger summary + the composite that lays out the three
regions (board · shelf · ledger) into one screen. **No I/O** — it takes the already-rendered/already-
read inputs and composes. Two parts:

1. **`homeLedgerLine(report)` — the foot (DL-6 / DL-8 / E-028).** A **pure** one-liner that renders the
   walk-away report (`WalkAwayReport`, `src/ledger/walk-away.ts`) as the compact trust line at the foot
   of Home: `ledger   E1 walk-away N% (k/n)   └ forward L% · attested A%` — the **provenance-split**
   E-028 already computes (forward·attested), NOT a single conflated rate. The full readout stays
   `vend audit` (DL-8) — this is the glance. **Honest-empty:** when there are no runs yet, it says so
   plainly (e.g. `ledger   E1 walk-away — no runs yet`) — **never a fabricated trust number** (the
   standing read-never-invent discipline; don't synthesize the E1 bit). Reuse the split sub-stats
   `formatWalkAwayFindings` (~238) already derives — mirror its labels so Home and `vend audit` read
   identically (no display drift), just on one line.
2. **`renderHome({ boardMenu, shelfRows, ledger })` — the composite (DL-6 / DL-1 / DL-9).** A **pure**
   layout-composer. It takes the **board** as the already-rendered `renderMenu` string (the board
   leads — column 0, unchanged), the **shelf** as `ShelfRow[]` (T-030-01) rendered *receding beneath*
   under a `shelf` heading, and the **ledger** as the `homeLedgerLine` string at the foot. Whitespace
   divides the three regions — **no boxes, no rules, no card chrome** (DL-1/DL-9). The shelf region is
   **receding inventory, not press-selectable** in this slice (the board owns the selection numbers) —
   render the shelf rows so they don't read as the press namespace (reuse `renderShelf`'s clean-
   typographic key — worth leads, envelope·confidence recede — but the Home shelf is informational).
   Empty board → the `renderMenu` guidance line passes through; empty shelf / empty ledger render their
   honest one-liners (the `renderShelf` / honest-empty precedent), never an error.

Return plain strings (the render shell's currency). Do **not** read files, gather, or recalibrate here
(T-031-02 does the I/O); do **not** change `renderMenu` / `renderShelf` / recalibration (compose them).
