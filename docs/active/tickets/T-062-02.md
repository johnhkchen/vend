## Context

Produce a single self-contained `vend` binary via `bun build --compile` for the pinned target that bundles BAML's ~108 MB native addon and runs with no repo checkout.

_Advances: P5, E-061: a self-contained per-platform binary, offline-usable on one machine_

## Acceptance Criteria

- [ ] On a directory with no node_modules/checkout, the compiled binary runs `vend --version` (real semver) and exercises a BAML-backed path proving the native addon loads.
