## Context

The visual tokens the whole charter composes from, in the human-set **clean-typographic** key. Start
`docs/knowledge/design-language.md` (DL-1…, the visual complement to `information-architecture.md` —
same capped, principle-level, anti-stale shape) and capture the **atoms**:

- **DL-1 — the governing principle: clean-typographic restraint.** Type hierarchy, weight/case, indent,
  and whitespace carry the information; chrome (boxes, rules) is the exception; **color is meaning, not
  decoration.** (The parallel to IA-1's governing decision.)
- **The palette / tone meanings** — what each tone *means*, not a hex spec: **amber = the andon, and
  nothing else** (IA-9, the single reserved saturated color); **default foreground** = content +
  in-flight; **dim/muted** = metadata, envelopes, secondary; a **quiet settle accent** for a clean
  finish; **never red, ever** (IA-9 — a stop is a successful refusal, not an error).
- **Type hierarchy** — how the recommendation leads and the shelf recedes (IA-1) using only the
  terminal's levers (case, weight, dim, indent, whitespace), since a terminal has little type to spend.
- **The two honest rules the atoms must encode** (these are *load-bearing*, not decoration):
  - **The meter (IA-8)** — ◇ tokens (a detect-after *burn*) and ⏱ wall-clock (a hard-wall *countdown*)
    are drawn **differently** so the meter cannot lie about its two denominations; state the visual
    distinction (e.g. a countdown vs a fill, or distinct glyphs/labels).
  - **The andon (IA-9)** — amber, calm/protective voice, carries its four things (which gate · what
    survived · why · the next pull); its typographic treatment (how it sets apart without red/chrome).

Ground the tokens in the **surfaces we already emit** — the `vend work` production line + receipt, the
`vend audit` split readout, the stale-board andon — so the charter formalizes the live look, not an
imagined one. Capped + durable; principle-level, not a pixel/component spec.

**Cites:** `docs/knowledge/information-architecture.md` (IA-1/6/7/8/9 — the principles being given a
look); `card-model.md`/`mana-economics.md` (lenses, scoped to docs); the live `vend work`/`vend audit`
output as the reference surface; `vision.md`. The chosen direction (clean-typographic, human-set) is
fixed, not re-opened.
