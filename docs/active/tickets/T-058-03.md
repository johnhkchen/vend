## Context

**Pre-wire the Astro seed to DRIVE with vend — the files that turn a frontend repo into a two-gesture vend
project.** Layer into `examples/templates/hackathon-seed/` (the brief's piece-B content), centered on the
designer's *visual* path:

- **`README.md`** — the copy → `vend init --template hackathon` → `vend doctor` → edit `SEED.md` →
  `vend steer` → review board → `vend work` drive script, with **the SVG board (the picture, not the code)
  as the centerpiece** of "what you'll see". A designer-legible script, not a dev README.
- **`SEED.md`** — one filled example idea (a small frontend feature for the Astro app, e.g. "a team-finder
  page that matches hackathon-goers by skill + idea overlap") + a one-line "replace this with your idea",
  + the pair (driver: dev; partner: designer). The seed is the ONLY input the user authors.
- **`charter.md`** — a hackathon-tuned value function: **a demonstrable runnable slice over polish**;
  *valuable* = advances a runnable/deployable demo; right-sized to a session; gates kept light but real.
  Teaches the general clearing move while honestly domain-specific.
- **`shelf-note.md`** — points the pair at the shipped articulation trilogy: `vend steer` (or `survey`) to
  read the seed → ranked board + the real forks; then `vend work` to clear the first slice on a budget.
- **`EXPECTED-OUTCOME.md`** — a *target* stub (the gold master is FILLED from the real live run in T-058-05):
  what a good drive yields — a coherent ranked board off the seed, a handful of genuine forks, ≥1 cleared
  slice. (This ticket writes the target shape; T-058-05 captures the actual outcome into it.)

Honest-empty held: these add structure + charter + SEED, NEVER demand (the overlaid board starts empty).

articulation plays the drive points at, with the **codebase-memory-mcp** (project

- `search_code(pattern: "charter|shelf|steer|survey|vend work|CLAUDE.md|value function", regex: true, mode: "full", path_filter: "^(src/play/(steer|survey|work)|docs/knowledge)")` — the charter/shelf shape + the steer/survey/work gestures the drive script and the tuned charter must match.

**Cites:** `examples/templates/hackathon-seed/` (the seed dir from T-058-02); `docs/knowledge/` (the charter
