## Context

The counter gesture that makes "fund it, walk away for two hours" a real command — the
Confirm→Run→Settle spine (IA-6) at macro scale. Wire the wallet (T-024-01) + the spend loop
(T-024-02) behind a CLI verb:

```
vend work --budget <ms>,<tokens>     # fund the macro wallet, spend it down on the ranked board
```

- **Confirm:** parse `--budget` into the macro `Budget`, allocate the wallet, echo what it will do
  (fund X, spend down the staged board by leverage rank) — the single funding gesture (P2). Pre-fill
  / default the budget from the board's tier where sensible (IA-6), adjust is the exception.
- **Run:** drive the loop, streaming the **production-line** signal (IA-7) — which pull is running,
  against the wallet burn (both denominations, IA-8) — never the raw executor stream.
- **Settle:** print the **receipt** (IA-6) — what cleared, what each cost, what the wallet has left,
  and **why it stopped** (exhausted / board cleared / andon — the andon rendered amber as a
  successful refusal, IA-9, never red).

Follow the existing `cli.ts` argument-parsing shape (the `run` / `audit` verbs as the pattern); the
parsing is pure + testable, the loop drive is the impure wiring. The session result T-024-02 returns
is the Settle's data — this ticket renders it, it does not recompute it.

**Cites:** `src/cli.ts` (the verb/arg-parsing surface to extend — mirror `vend run` / `vend audit`);
T-024-02 (the loop + its structured session result) + T-024-01 (the wallet readout `formatWallet`);
`information-architecture.md` IA-6 (Confirm→Run→Settle), IA-7 (production-line, not raw stream), IA-8
(two-denomination meter), IA-9 (andon amber = successful refusal); P2 (the funding gesture).

## Acceptance Criteria

- [ ] `vend work --budget <ms>,<tokens>` parses + allocates the wallet (pure arg-parsing, **unit-
      tested** like the existing verbs — valid/invalid budget, defaults); a malformed budget is a
      clean usage error, not a crash.
- [ ] Run streams a node-level production-line signal (which pull, against the wallet burn — IA-7/8),
      not the raw executor stream; Settle prints the receipt (cleared / per-cast cost / remaining /
      **stop reason**, the andon amber per IA-9).
- [ ] End-to-end honest: the gesture funds once, the loop spends down, the hard-stops hold (P7) — a
      live `vend work` against the staged board clears at least one cast and Settles truthfully.
- [ ] `bun run check:*` green.
