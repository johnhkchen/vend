## Context

The headline: cast a play, and its envelope is **proposed from its own measured past**.
A **pure** `recalibrate(play, records, tier)` bounds a play's cost — both tokens and
wall-clock — at the **value-tier percentile** (IA-12: Keystone→~p95, Standard→~p90,
Leaf→~p75), computed from **successful** runs only (IA-13). It returns the measured
envelope **plus confidence** (N successful runs, censored/andon count). Then surface it:
the CLI / Confirm default shows the **measured** envelope with honest confidence,
falling back to the hand prior when there's no data (cold start).

**Estimator:** an **exact percentile over the windowed records** — NOT t-digest (that's
a scale optimization; the log is small — ship the smaller real thing). Censored
(andon'd-at-envelope) runs are **not** averaged in; they are lower bounds (IA-13). This
slice **reads** the andon rate into the confidence; it does not yet auto-tune or act on
it (IA-14 is a later rung).

**Cites:** `src/log/run-log.ts` + the reader (T-013-01); `src/budget/budget.ts` (the
`Budget` it returns); `src/cli.ts` / `src/shelf/press.ts` (the default surface);
`information-architecture.md` IA-6 (Confirm default), IA-8 (honest confidence),
IA-12/IA-13 (percentile from successes).

## Acceptance Criteria

- [ ] **Pure** `recalibrate(play, records, tier)`: exact value-tier percentile over
      **successful** runs' tokens + wall-clock → a `Budget`; returns
      `{ envelope, confidence: { successes, censored } }`; **cold-start fallback** to a
      hand prior when successes are below a small threshold (e.g. < a handful).
- [ ] A surface (CLI flag/output and/or the Confirm default) shows the measured envelope
      labelled with confidence — "measured · N casts · pXX" vs "estimate (no data)".
- [ ] Unit-tested with a fixture log: correct percentile per tier; censored runs excluded
      from the percentile but counted; cold-start falls back to the prior; tokens and
      wall-clock bounded independently.
- [ ] `bun run check:*` green. (The live check — cast a play, see its envelope proposed
      from history — is the human verification at sweep.)
