## Context

E-022's equivalence judge compared **whole survey boards** and read `genuine-disagreement` (0.00,
N=3). But a board can disagree in the *tail* while agreeing on the *head*, and the contract's real
concern is the **load-bearing #1 pull** (IA-1's single recommendation). Build the **head-isolating**
measurement: given a play's N collected survey boards, extract each board's **#1 (and top-k) ranked
signal** and classify whether the *head* is stable across the N casts **independently of tail
order**.

Split per the house pattern (exactly as `equivalence.ts` / `run-equivalence-judge.ts`): a **pure
core** — given the N boards' extracted top-picks (or the per-pair head-equivalence verdicts), return
a `head-stable | head-flips | mixed` classification + a score, **unit-tested** on fixtures; the
impure harness — parse the staged survey board markdown to pull the ranked signals, reuse the
equivalence judge to compare *just the heads*, and print the head verdict **beside** E-022's
whole-board dispersion — is not unit-tested (the live-cast / fs / dispense verbs).

The survey board's shape is fixed (a ranked markdown table + a "Pull these" block, top row first —
see a sweep board in `work/T-022-02/sweep-logs/` and `src/play/survey.ts`); the parser keys on that
ordering to extract the #1 / top-k.

**Cites:** `src/probe/equivalence.ts` + `src/probe/run-equivalence-judge.ts` (T-022-01 — the core +
harness to extend, the judge-as-comparator); `work/T-022-02/findings.md` (the head-vs-tail caveat
this measures); `src/play/survey.ts` (the board shape the parser reads); a disposable temp ledger
(the probe discipline — no real-log pollution).

## Acceptance Criteria

- [ ] A **pure** core that, given N survey boards' top-picks (or per-pair head-equivalence
      verdicts), classifies head stability as **head-stable / head-flips / mixed** + a summary
      score; **unit-tested** on fixtures (all-same-#1 → head-stable; all-different-#1 → head-flips;
      a mix → mixed; tail re-order with a stable #1 → head-stable).
- [ ] A harness that collects N survey boards (reusing the equivalence probe's seeding/collection),
      extracts each board's ranked #1/top-k, casts the equivalence judge over **just the heads**,
      and prints the `head-stable | head-flips | mixed` verdict **beside** the existing whole-board
      dispersion + equivalence numbers.
- [ ] `bun run check:*` green; the E-022 consistency + equivalence probe paths are unaffected
      (extend, don't break).
