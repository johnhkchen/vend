## Context

The pure foundation of the supply shelf: each playbook's **worth** (role) + its **warranted budget**
(recalibrated envelope from actuals, E-013). Two parts:

1. **Worth on the contract.** Add `readonly summary: string` (a one-line "what this playbook is for")
   to the `Play` interface (`src/engine/play.ts`) and set it on the six registered plays
   (decompose-epic, expand-fragment, capture-note, propose-epic, steer, survey) — short, role-level
   (e.g. survey → "read the project → a ranked demand board"). Optional vs required: make it
   **required** so every play declares its worth (the six are all in-repo; a missing one is a compile
   error, which is correct — an unworthed play shouldn't ship).
2. **The warranted-budget core.** A **pure** `shelfRows(plays, records)` that, for each play, pairs
   its `summary` with its **recalibrated envelope** (`recalibrate(play.name, records, tier, play.budget
   as the cold-start prior)`, E-013) and a **confidence** read lifted from `recalibrate`'s result —
   `measured (N runs)` when the play has history, `default` when N=0 (cold-start to the authored
   `play.budget`, labelled as default, **never a guess dressed as measured** — the E-026 lesson). No
   I/O: it takes the already-read records.

Return a typed `ShelfRow[]` (name · summary · envelope · confidence) for T-030-02 to render. Do not
edit recalibration math (E-013 owns it); do not add value tiers to plays (a play's worth is its role,
not a leverage tier).

**Cites:** `src/engine/play.ts` (`Play` ~127 — add `summary`; `registry` — enumerate the six);
`src/ledger/recalibrate.ts` (`recalibrate` + its `confidence` — sample size / censored / cold-start);
`src/log/run-log.ts` (`RunRecord`/`forPlay` — the actuals, passed in); `src/shelf/gather.ts`
(`budgetForTier`/`Budget` — the tier for the recalibration percentile); the six `src/play/*.ts`.

## Acceptance Criteria

- [ ] `Play` gains `readonly summary: string`, set on all six registered plays (role-level, one line);
      `tsc` proves every play declares it (the cast/registry paths unaffected).
- [ ] A **pure** `shelfRows(plays, records)` returns one row per play with `summary` + the
      **recalibrated** envelope (E-013, cold-start to the play's authored `budget` when N=0) + a
      **confidence** field distinguishing `measured (N)` from `default` (no runs).
- [ ] **Unit-tested:** a play with history → measured envelope + N; a play with no records → the
      authored default, labelled `default` (not measured); the row carries the worth verbatim. `bun
      run check:*` green.
