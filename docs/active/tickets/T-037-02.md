## Context

**THE proof — a watched, metered macro-wallet spend.** This is the live cast the project has
deferred since T-024-03 ("the full live multi-cast spend is the one deferred human sweep"). T-037-01
gave the go (auth==exec holds, the bounded run will clear). Now run it for real.

> **This ticket performs a LIVE, METERED spend** — real `claude -p` casts via the chain, real
> tokens, real wall-clock (minutes–tens), and it **mints real epics+tickets** onto the board (the
> cleared pulls). The human running `lisa loop` authorizes the spend; the **budget bounds it** (P7).
> This is NOT a free deterministic proof — the deliverable *is* the watched run.

Steps:

1. **Stage a FRESH board (live cast).** `DEFAULT_BOARDS` were removed (2026-06-20), so stage one:
   `bun run src/cli.ts steer` (writes `docs/active/pm/staged/steer.md` — board + forks; the
   DEFAULT_BOARDS[0]) or `… survey`. A board staged now is newer than `docs/active/**`, so it clears
   the E-027 freshness gate. (steer/survey are themselves real casts — bounded by their own
   envelopes.)
2. **Fund a BOUNDED budget and walk away.** The human picks `--budget <ms>,<tokens>` at the counter
   (P2) — **recommend ~1h/~1M tokens** (`--budget 3600000,1000000`), a real but contained spend (≈2
   chains), NOT the 2M default. Run: `bun run src/cli.ts work --no-intervened --budget 3600000,1000000`.
   `--no-intervened` is the walk-away test (let it clear; the records are genuine forward E1). Let it
   spend down — pull the highest-leverage affordable signal, cast propose→decompose, debit the wallet
   by the ACTUALS, repeat — until a **clean stop**: wallet-exhausted or board-cleared.
3. **Capture the evidence** to `docs/active/work/T-037-02/sweep-log.md`: the production-line steps
   (the `onStep` emits), the **receipt** (`renderReceipt` — funded vs spent per denomination, the
   clean stop reason), the **cleared pull(s)** (the real epic id(s)+tickets minted — run `lisa
   validate` and confirm green on them), and the **forward-E1 record(s)** appended to
   `.vend/runs.jsonl` carrying `intervened:false`. Paste the actual terminal output, not a paraphrase.

**P7 is the contract:** the wallet debits real actuals and stops clean — **nothing partial**, a
truthful receipt, no cast the wallet couldn't afford. If it clears 0 (price > funded, or a gate
andon), that is an HONEST outcome to record, not a failure to hide — but T-037-01's go should have
prevented a price-mismatch no-op.
