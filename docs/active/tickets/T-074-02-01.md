## Context

A pure, fixture-tested decision that says whether a funded budget falls far enough below a play's measured envelope floor to warrant a warning, and produces the warning text — the reusable heart of the counter-time check.

_Advances: P7 — Budget is a hard contract_

## Acceptance Criteria

- [ ] A pure `underfundingWarning(funded, floor)` returns a warning message naming funded vs floor IFF funded tokens fall below the measured floor by the chosen factor, and returns none when funded ≥ (or near) the floor; unit-tested across the field-report ratio (12.5k vs ~400k → warn) and an adequately-funded case (→ silent), with no I/O and no addon load.
