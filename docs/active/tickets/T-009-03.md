## Context

Wire ProposeEpic as a registered `Play` and cast it — the parallel of `capture-note`
(read `src/play/note.ts` and mirror it). `src/play/propose-epic.ts`:
`proposeEpicPlay = { name: "propose-epic", render (b.request.ProposeEpic +
extractPromptText), parse (b.parse.ProposeEpic), gates (propose-core), effect (write
the minted E-0XX.md under the project root), budget, card (Blue Permanent) }`,
self-registered on the engine; `assembleProposeEpicInputs` (the demand signal text +
`charter.md` + a project snapshot, mirroring `assembleNoteInputs`); and a
`castProposeEpic` entry. Prove ProposeEpic casts through the **same** `castPlay`.

**Pull-discipline (PE-1):** the cast is invoked deliberately on a *pulled* signal —
never an auto-drainer over the board.

**Cites:** `src/play/note.ts` (`captureNotePlay` / `assembleNoteInputs` /
`castCaptureNote` / self-register — the exact pattern); `src/engine/play.ts` +
`cast.ts` (registry + `castPlay`); `src/baml/decompose-bridge.ts` (`extractPromptText`).

- [ ] `proposeEpicPlay` is registered; `castProposeEpic({ signal, budget, … })`
