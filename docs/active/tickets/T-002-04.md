## Context

Prove the slice dispenses something real: run the play **live** on an actual epic
and confirm the whole loop — gated `WorkPlan`, materialized lisa-valid files,
countable log, budget honored, and a deliberately-broken input that **stops the
line**. The bootstrap payoff: the play can now do by machine what we did by hand to
produce E-001. A natural live input is **E-002** (the CI epic, currently a demand
signal on the pull board) or a small fixture epic.

## Acceptance Criteria

- [ ] A live `vend run decompose-epic` on a real epic produces stories/tickets that
      pass `lisa validate`, with a run record appended to `.vend/runs.jsonl`.
- [ ] A run given a tiny budget hits the budget andon and stops cleanly — **no
      partial materialization**.
- [ ] A deliberately under-specified epic trips a clearing gate and stops the line
      with a named reason — no garbage emitted.
- [ ] A short note records tokens / cost / wall-clock of the real run, and any gap
      between the by-hand E-001 decomposition and the machine output (the first
      kaizen signal for the play).
