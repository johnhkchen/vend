## Context

**Clamp the per-cast funding guard to a rational token BAND `[350_000, 700_000]`.** `fundingEnvelope`
(`src/ledger/recalibrate.ts`, E-050) computes the envelope a cast RUNS under: cold-start / high-censored ⇒
`max(priced, maxCensoredActual × MEASUREMENT_HEADROOM)`, else `priced` (a well-calibrated play funds at its
bare p90). Both extremes are now irrational: the bare p90 is **too tight** (well-calibrated propose funds at
~170k and budget-exhausts on a 3.6% tail — the `vend chain` halt), and the censored-widen is **unbounded**
(E-051's decompose self-funded to ~733k). Add a floor and a hard ceiling.

**The change (in `fundingEnvelope`, beside `MEASUREMENT_HEADROOM`/`CENSORED_WIDEN_RATE`):**
- `export const FUNDING_FLOOR_TOKENS = 350_000;` — below this we don't care; every cast is funded at least
  this so a too-tight p90 never starves a real cast.
- `export const FUNDING_CEILING_TOKENS = 700_000;` — the hard P7 wall; no cast is ever funded beyond this
  (runaway self-funding is rejected here).
- Clamp the function's **TOKEN** output: `tokens = min(FUNDING_CEILING_TOKENS, max(FUNDING_FLOOR_TOKENS,
  computed))` — applied AFTER the existing `max(priced, maxCensoredActual × headroom)` step, so the band is
  the outermost bound. (Overridable via the existing `opts` for tests, like the other knobs.)

**Strict scope (do NOT touch):** the **WALL-CLOCK** dimension (the policy is tokens; time keeps its E-038
headroom — clamp tokens only); `recalibrate`'s returned `envelope`, the percentile math, or
`formatEnvelopeLabel` (the QUOTED price stays the honest p90 — IA-8; the band is on the GUARD, not the
price); `canAfford`/`fitNext` (authorization gates on price, unchanged). The `widened` flag stays whatever
the existing logic sets (the floor/ceiling are bounds, not the headroom signal). PURE/TOTAL, positive-int
(reuse `positiveInt`), the constants finite (P7).

**Codebase-memory (use it — project coding-task suggestion):** confirm `fundingEnvelope` + the GUARD-≠-PRICE
seam with the **codebase-memory-mcp** (project `Volumes-ext1-swe-repos-vend`, indexed/ready) via
`search_code`, not raw grep:
- `search_code(pattern: "fundingEnvelope|MEASUREMENT_HEADROOM|CENSORED_WIDEN_RATE|FundingResult|positiveInt|maxCensoredActual", regex: true, mode: "full", path_filter: "^src/ledger/recalibrate")` — the funding core to clamp (the band is the outermost bound on its token output).

**Cites:** `src/ledger/recalibrate.ts` (`fundingEnvelope` — add the floor/ceiling constants + clamp its
token output; `positiveInt`; leave `recalibrate`/`formatEnvelopeLabel` untouched); `src/play/work.ts:194-199`
(the GUARD ≠ PRICE seam the band respects); `docs/active/epic/E-053.md`.
