## Context

Render T-030-01's `ShelfRow[]` as the supply shelf and surface it behind a `vend shelf` verb — the
clean-typographic supply view beside the board (IA-2). Two parts:

1. **The render (pure).** A `renderShelf(rows)` in the design-language key (E-029 **DL-6** the shelf,
   **DL-9** no card chrome, **DL-3** type hierarchy): a flat numbered/dashed list, each row
   `name   summary   envelope · confidence` — worth leading, the warranted budget + confidence
   receding (dim/indent per DL-3), **no boxed cards**, no color except where an andon would apply
   (there is none here). Reuse `formatBudget` (`src/shelf/menu.ts`) for the envelope so the shelf and
   board read identically. A `default`-confidence row says so plainly (e.g. `~2h/50k (default — no
   runs yet)`); a measured row shows `(measured · N runs)`.
2. **The `vend shelf` surface (impure).** A CLI verb that gathers the registry's plays + reads the run
   log (`readRuns` over `.vend/runs.jsonl`), calls `shelfRows` → `renderShelf`, prints it. Mirror the
   existing `vend audit` read-only verb (parse path + dispatch arm); no budget/flags needed (it's a
   read). Add it to the usage line.

Keep the board (`vend`) untouched — this is the supply view *beside* it, not a board change.

**Cites:** T-030-01 (`shelfRows` + `ShelfRow`); `src/shelf/menu.ts` (`renderMenu`/`formatBudget` —
the board render to mirror in the shelf's flat key); `src/cli.ts` (`parseArgs` + a read-only verb arm
like `audit`; the usage line); `design-language.md` DL-6 (shelf beneath board), DL-9 (no cards), DL-3
(hierarchy); `src/log/run-log.ts` (`readRuns`). The live run log is the proof input.

## Acceptance Criteria

- [ ] A **pure** `renderShelf(rows)` prints the supply shelf clean-typographic (DL-6/DL-9/DL-3): flat
      rows, worth leading, warranted budget + confidence receding, **no boxed cards**, `default` vs
      `measured (N)` shown honestly; **unit-tested** on fixture rows (measured + default).
- [ ] A `vend shelf` verb gathers the registered plays + the run log and prints the shelf; the board
      (`vend`) and other verbs are unaffected; usage line updated. Parser arm covered like `audit`.
- [ ] **Live proof (free, no cast):** `bun run src/cli.ts shelf` lists the six playbooks with their
      worth + warranted envelopes — propose/decompose `measured (N)`, thinner plays honestly labelled,
      any zero-history play `default`. `bun run check:*` green.
