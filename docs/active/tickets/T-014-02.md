## Context

The **E2 / consistency** arm (PRD KR3) — the core promise, never measured. Run **one
playbook 5× with gates and 5× without** on a fixed input; diff the materialized output;
report the **gate-driven variance reduction** as a single number (gated should vary
materially less than ungated).

Enabler: there is no gates-off path today. Add the **minimal** one — a `--no-gates` run
mode (skip the gate phase in `castPlay`), or, if cleaner, compare against the raw dispense
output. Keep it minimal — a switch, not a framework (PRD §7).

**Anti-scope-creep:** ≤ this one paired experiment; do not generalize into a benchmarking
harness. Independent of E1 (T-014-01) — runs in parallel.

**Cites:** `src/engine/cast.ts` (`castPlay` — the gate phase to bypass);
`src/gate/gates.ts` (the gates toggled off); a small variance/diff over materialized
output; the PRD KR3 + §7.2 feature 3.

## Acceptance Criteria

- [ ] A minimal `--no-gates` run mode (or equivalent raw-dispense comparison) that runs a
      play with the gate phase skipped; the normal gated path is unchanged when the flag is
      absent.
- [ ] A variance harness: run one play 5× ±gates on a fixed input, diff the materialized
      outputs, and emit a **single variance-reduction number** + the raw per-run diffs.
- [ ] The pure diff/variance computation is unit-tested on fixture outputs; the live 5×2
      run is the human step at sweep (≤ the PRD's cast budget).
- [ ] `bun run check:*` green; the gated path and existing casts unaffected.
