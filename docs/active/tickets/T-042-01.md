## Context

Pure check/report model — a Check (name, ok, fix-it hint) plus a pure renderer mapping a set of probe results to a human-readable report and an exit code (0 when all green, non-zero when any check fails).

_Advances: P3, E-042 clean gated refusal_

## Acceptance Criteria

- [ ] Unit tests: an all-green result set renders each dep green and yields exit code 0; a set with any single failing check renders that check's name plus its fix-it hint and yields a non-zero code; the module performs no probing or IO (pure, mirrors the *-core split).
