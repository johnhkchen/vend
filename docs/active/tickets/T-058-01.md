## Context

**`vend init --template <name>` — overlay a named template onto the base scaffold (PM brief piece A; the
reusable seam every example reuses).** Today `vend init` writes the base `SCAFFOLD_MANIFEST` (write-if-absent,
never clobber, refuses if not a lisa project, board starts empty — E-040). Extend it:

- Add an optional `--template <name>` arg to the `init` CLI parse (`{ cmd: "init", template?: string }`).
- After the base scaffold applies, **overlay the named template's files** through the SAME converge planner
  (write-if-absent, never clobber, reported as DATA). House split: a pure `planTemplate(existing, base,
  overlay)` decides the plan; the effect writes (mirror `init-core.ts` / `init-effect.ts`).
- **Unknown template name → a clean refusal** with the list of available templates (mirror E-040's
  `not-lisa` refusal shape: DATA + fix-it hint + non-zero exit).
- **Honest-empty invariant held:** the overlay adds structure + a tuned charter + a `SEED.md`, **NEVER
  demand** — the board still starts empty; the first move is a cast (IA-3/IA-4). One-way-to-lisa held (the
  template creates only vend-owned paths). Bare `vend init` (no `--template`) is byte-identical to today.

(The hackathon template *content* is T-058-02/03; this ticket is the seam + a trivial template registry so
`--template hackathon` resolves.)

**Codebase-memory (use it — project coding-task suggestion):** confirm the scaffold core + effect + the
init CLI parse with the **codebase-memory-mcp** (project `Volumes-ext1-swe-repos-vend`, indexed/ready) via
`search_code`:
- `search_code(pattern: "SCAFFOLD_MANIFEST|planScaffold|ScaffoldEntry|ConvergePlan|init|template|not-lisa|refus", regex: true, mode: "full", path_filter: "^src/(init|cli)")` — the pure converge planner + the write effect + the init arg parse + the `not-lisa` refusal shape to mirror.

**Cites:** `src/init/init-core.ts` (`SCAFFOLD_MANIFEST` + the converge planner — add `planTemplate` overlay);
`src/init/init-effect.ts` (the write effect); `src/cli.ts` (the `init` arg parser — add `--template`; the
refusal/usage shape); E-040 (the base scaffold); `docs/active/epic/E-058.md`.
