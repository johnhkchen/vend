## Intent — the bigger-picture play

`information-architecture.md` settled **what the user sees and how they move**; this settles **what it
looks like** — the concrete visual surface the eventual TUI implements, so a TUI epic builds a decided
look instead of re-litigating it. The human set the direction: **clean typographic.** Terminal-native
restraint — type hierarchy, weight/case, indent, and whitespace carry the information; chrome (boxes,
rules) is the exception; **color is meaning, not decoration.** The single saturated color is **amber,
reserved for the andon** (IA-9); everything else is the terminal's default foreground + a dim/muted
secondary, with no red, ever. The MTG **card/mana model stays a lens** — it lives in `card-model.md`
and the specs (and these epic cards), but the TUI does **not** render boxed cards; cost may appear as
a compact inline glyph at most. This epic captures that as **DL-1…DL-N** — a capped, anti-stale spine,
exactly the shape of the IA doc.

## Value to the design

Gives every future TUI epic a settled visual contract, so they implement rather than invent — the same
leverage the IA spine gave navigation. It makes the two most design-laden invariants legible and
honest: **P3** (the andon — gates-are-the-contract — rendered as a *successful refusal*, amber not red,
IA-9) and **P7** (the budget — rendered as the two-denomination meter that *must not lie*, IA-8: a
hard-wall countdown vs a detect-after burn, never two identical bars). Restraint is not austerity here
— it is the legibility a walk-away tool needs (you glance, you trust, you leave).

## Done looks like

A capped `docs/knowledge/design-language.md` (DL-1…DL-N, the visual complement to the IA spine) that a
TUI epic can implement without a design decision left open:

- **The governing principle** — clean-typographic restraint: type/whitespace carry it, color is
  meaning, chrome is the exception.
- **The palette** — what each tone means: amber = andon (reserved); default foreground = content +
  in-flight; dim/muted = metadata/envelopes; a quiet settle accent; **never red** (IA-9).
- **Type hierarchy** — how the board's "NOW + why" leads (IA-1) and the shelf recedes, using case /
  weight / dim / indent (the terminal's limited type, used deliberately).
- **The meter** (IA-8) — ◇ tokens (detect-after burn) vs ⏱ wall-clock (hard wall) drawn *differently*,
  so the meter cannot lie about its two denominations.
- **The production line** (IA-7) — node-level progress + one distilled "what it's doing" line, never
  the raw executor stream.
- **The andon** (IA-9) — amber, the four things it carries, the calm/protective voice, its typographic
  treatment.
- **The surfaces** — Home (board + shelf), the Counter (Confirm→Run→Settle, IA-6), the Ledger — each
  sketched in the clean-typographic key with a small ASCII reference mock.
- **The card-as-lens decision** — stated explicitly: cards are a doc/spec lens, not TUI chrome.

## Context & constraints

- **Cites:** `docs/knowledge/information-architecture.md` (IA-1 the recommendation-first home, IA-6
  the Counter spine, IA-7 the production line, IA-8 the two-denomination meter, IA-9 the amber andon —
  the principles this gives a *look*); `card-model.md` + `mana-economics.md` + `tps.md` (the lenses,
  now scoped to docs not chrome); `vision.md` (the v1 TUI surface); the chosen direction (clean
  typographic, human-set). **Grounds in** the actual rendered surfaces we already emit — the `vend
  work` production line + receipt, the `vend audit` readout, the stale-board andon — as the live
  reference the charter formalizes.
- **Capped + durable (anti-stale):** principle-level, small, like the IA doc; not a pixel spec, not a
  component library. Where a future TUI and this charter disagree, fix one — no drift (the IA doc's
  own rule).
- **Non-goals:** building any TUI (a downstream epic implements the charter); a full component/widget
  library or theming engine; re-opening the direction (clean-typographic is **set**); a color-accessible
  audit (a downstream refinement); re-deriving the IA (this is its visual complement, not a revision).
  Not an executor (**N4**).
- **First slice = the captured charter** (`design-language.md`, DL-1…DL-N) in the clean-typographic
  key, with reference mocks. **Out:** any implementation.
