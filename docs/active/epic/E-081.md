## Intent — the bigger-picture play

The 2026-07-13 steer jank ledger (items 2-3, the pair its own disposition calls 'the sharp ones') caught the flagship gesture speaking three token/turn dialects at once: the summary line honestly prints agent turns, the durable `turnsUsed` field records executor conversation events (2, 20, and 45 for comparable casts — 45 on a 15-capped dispense), and the live progress line's final token figure read 105k/400k while the ledger's true weighted spend was ~214k. Make the record and the line speak the meter's units: record deduped agent turns (the capped unit) in the ledger, keeping the event count only as its own honestly-named field if it earns its place, and root-cause the 45-in-a-15-cap anomaly while in that path (working hypothesis: sidechain/subagent assistant messages inflating the dedup count); diagnose which buckets or message kinds the live accumulator's fold misses, make the live figure track true weighted spend within honest tolerance, and label what it is. This matters now because these are the exact fields recalibration and envelopes read — every run they mix units, the misallocation E-068 existed to end creeps back in through the measurement layer itself.

## Value to the design

Restores the meter's honesty end-to-end (P7 — a line that reads half the real spend invites exactly the misallocation the budget contract exists to prevent), and finishes the lisa field report's measurement-vocabulary ask (P3): one unit, meter to line to ledger. T-072-04-01 fixed only the display third of that sentence; this card fixes the durable record that audits and envelope math depend on, and the live number the operator actually allocates by.

## Done looks like

On a fresh cast from the installed binary: the final live progress line's token figure agrees with the ledger's weighted spend within a stated, labeled tolerance (no silent ~2× undercount, and the label says what the number is); the new run's ledger turn field records deduped agent turns matching the summary line, with any retained event count under its own honestly-named key; and the 45-in-a-15-capped-dispense anomaly has a written, evidence-backed root cause. Historical ledger lines are untouched, their old unit noted rather than laundered.
