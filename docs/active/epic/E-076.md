## Intent — the bigger-picture play

v0.4.0-rc.4 shipped E-073's cross-vendor review gate with a fatal default: the settlement path
resolves a complement reviewer from a default registry that fabricates a codex reviewer on
`DEFAULT_OPENAI_BASE_URL = http://localhost:11434/v1` (Ollama's port), and the verdict dispense is
uncaught on failure. Because E-073's diff capture runs on every cast and E-071 stamps
`seatOfExecution`, the "deliberately inert for a one-seat registry" intent written on the gate
never triggers — so every rc.4 metered cast with default config crashes at its first settlement
with a raw ConnectionRefused stack. Field-confirmed twice within hours of publish (this repo's
E-075 chain; boilerplate-demo's E-013 chain): half-minted orphan epics, a spent cast with NO
ledger line, and a stray `.vend/artifacts/run-….diff` with no matching ledger row. Neither
operator had any reference to Ollama or port 11434 anywhere.

The ruling (2026-07-13, house precedent E-070 — preference degrades, refusal is for structural
invalidity): cross-review is INERT unless a reviewer seat is genuinely provisioned, with an honest
`cross-review-skipped` marker; a provisioned reviewer that fails at dispense time lands as the
existing amber andon (E-074) with a ledger line, never a crash; fail-closed stays available only
as explicit operator opt-in.

## Done looks like

On a machine with default config and NO network listener on 11434, a metered cast completes
end-to-end with cross-review inert and a `cross-review-skipped` marker on the run record — pinned
by a no-network characterization test (the exact field-failure shape). With a provisioned but
unreachable reviewer, the cast lands as a named andon outcome WITH a ledger line and no stack
trace. The run record and any captured diff artifact land together even when settlement throws.
`vend doctor` goes red with a named check when a reviewer would resolve but cannot dispense.
E-075's decompose (currently impossible) succeeds on the fixed binary — the orphan resolves as
the fix's live proof.
