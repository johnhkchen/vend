## Intent — the bigger-picture play

Today a cast is a single transaction: pick a play, allocate one envelope, run. The **founding
promise** is bigger — *"allocate a feature-block budget and walk away for two hours."* This epic
builds that: a **depleting wallet** the human funds once at the counter, and an **autonomous
spend loop** that pulls/chains casts against it, **fitting each next cast into the remaining
wallet using its measured price** (E-013's recalibrated, reference-class envelopes — not a guess),
and stopping cleanly when the wallet can't afford the next cast, the board is cleared, or a gate
fires. It is the literal realization of **P2** (two gestures: fund + walk away) at *macro* scale,
**P4** (autonomy — Vend is built to be left), and **P7** (the budget is a hard contract that
*depletes*).

This is deliberately the rung **above** E-013: you cannot spend a macro budget intelligently
against *guessed* per-cast costs. E-013 made the per-cast prices measured; E-024 spends a wallet
down against them. And per E-014's honest caveat, the wallet's own autonomous runs become the
**forward, variance-bearing E1** the back-fill sprint structurally couldn't be — every cast it
clears unattended is a live walk-away the human can self-report on.

## Value to the design

Turns the charter's headline gesture from a slogan into runtime behavior, and does it **safely**:
the wallet is *hard* (P7) — the wall-clock is a wall (halts mid-flight, IA-8), the andon stops the
line on a gate (IA-9), and the token meter depletes the wallet detect-after. Autonomy capitalized
on E-014's *didn't-break* trust evidence, **with the hard-stops intact** — not on faith. The
Counter's Confirm→Run→Settle (IA-6) finally has a macro-scale transaction to host.

## Done looks like

- **A depleting wallet:** the human funds a macro budget once (`vend work --budget <ms>,<tokens>`);
  each cleared cast **debits the wallet by its actual cost**; `remaining` is always truthful across
  the two denominations (wall-clock hard wall + token burn, IA-8 — never drawn as one bar).
- **An autonomous spend loop:** Vend pulls/chains the board's ranked work, **fits each next cast
  into the remaining wallet at its measured price** (E-013 `recalibrate.ts`), casts it, debits the
  actual, and **stops cleanly** on {wallet can't afford the next cast · board cleared · andon}. No
  overspend past the wall (P7); a gate-stop ends the session as a *successful refusal* (IA-9), not a
  crash.
- **A Settle summary (IA-6):** what cleared, what each cost, what the wallet has left, why it
  stopped — the receipt for a two-hour walk-away.

## Context & constraints

- **Cites:** `src/budget/budget.ts` (`Budget`/`Usage`/`check` — the per-cast envelope the wallet is
  built over); `src/ledger/recalibrate.ts` (E-013 — the measured, reference-class per-play *price*
  the loop fits the next cast against); `src/engine/cast.ts` (`castPlay`/`RunSummary` — the cast the
  loop drives; **`RunSummary` must surface the cast's actuals** so the wallet can debit precisely —
  today it returns outcome/materialized only, a small honest seam extension); `src/engine/chain.ts`
  (`castChain` — the pull→clear the loop repeats); `src/play/survey.ts` (the ranked board the loop
  pulls from); `src/log/run-log.ts` (where actuals already land — the alternative debit source); the
  Counter spine IA-6 + the two-denomination meter IA-8 + the andon IA-9.
- **Hard-stops are non-negotiable (P7):** the wall-clock halts mid-flight; the wallet never authorizes
  a cast it can't afford; an andon ends the session. The wallet *depletes* — a spent wallet is spent.
- **Honest scope:** first slice spends down the **already-staged board** (pull ranked signals in
  order, clear each via chain) — *not* an open-ended planner that invents new work. The "what to pull
  next" policy is the board's existing leverage rank (IA-1), not a new optimizer.
- **Non-goals:** a smarter pull-selection optimizer / lookahead (the leverage rank is the policy —
  a downstream pull if needed); parallel/fleet spending (one line at a time — the multi-DAG andon
  board is its own epic); a GUI budget dial (the CLI gesture is the v1 surface); re-deriving per-cast
  prices (E-013 owns them). Not an executor (**N4**).
- **First slice = the wallet + the loop + the `vend work` gesture** (~1 block). **Out:** optimizer,
  fleet, GUI.
