## Intent — the bigger-picture play

E-019's measured sweep showed the honest-empty gate is a false-negative machine on grounded input — survey abstained on 2/3 casts of a demand-rich board (67%) and expand on 1/3 casts of an evidently-grounded fragment (33%), declaring 'no demand' about input that demonstrably grounds real demand. A gate that censors valid work is worse than no gate: it silently swallows the articulation plays' output and makes their result a coin-flip on identical input. The play NOW is to recalibrate the abstention threshold so honest-empty fires only on input that is genuinely thin, restoring P3 (gates are the contract) for the survey and expand plays the rest of the shelf depends on.

## Value to the design

Makes the IA-4 honest-empty gate trustworthy: it stops over-firing on grounded demand while still abstaining on truly empty input. This directly advances the core feature (the clearing house must reliably turn real intent into staged work, not drop it) and hardens P3 — a gate is only a contract if its verdict tracks reality, and today it does not.

## Done looks like

Re-running E-019's consistency probe (`run-consistency-probe.ts`) on the same fixed grounded inputs, honest-empty no longer fires as a false negative — survey's demand-rich board stages signals rather than the no-demand marker, and expand's grounded fragment stops abstaining — driving the grounded-input honest-empty rate to ~0; AND a deliberately thin/empty input fixture STILL yields honest-empty, demonstrating the gate was tightened, not disabled. Both polarities verified by the existing probe harness.
