## Intent — the bigger-picture play

**User story:** a designer attends a hackathon; the dev says *"download vend + lisa, clone the frontend repo"* — an Astro + React template that auto-deploys to Cloudflare. The designer runs two gestures and gets a **visually satisfying understanding** of what's being built — the work-graph board, beside the app they're already previewing — instead of being knee-deep in code whizzing by.

This is the **Set-B non-dev channel proof** the PM desk flags as the riskiest, highest-value untested thing (`product-strategy.md`, `OKRs` Set B, `process-gate.md`). The whole recent build went deep on the engine/graph substrate and the visual surface (E-046→E-057); what's never happened is a *real designer driving a thin, non-vend domain*. This epic is that test: it instantiates the build-ready hackathon brief (`pm/brief-hackathon-example.md`: `vend init --template` + a `hackathon-seed/` template + a gold-master) as a concrete **Astro + React + Cloudflare** frontend seed, and makes the **SVG board (E-055/E-056) the designer's window** — served into the browser they already have open for the app. Maya (the designer persona, the originating "mini-stroke at text-heavy views" inbound) is the protagonist; the value is *seeing* the work, not reading the code.

## Value to the design

Realizes the OMTM the strategy names — *time-to-first-driven-board for a non-dev* — as a runnable artifact, and buys down the riskiest hypothesis on the desk (claim 2b: "non-devs want a visual surface") with a real drive instead of more surface. Advances P2 (the two-gesture transaction made onboardable: copy → init → steer → work, no code), P5 (local-first + the visual understanding; Cloudflare deploy is the *designer's* push, not a vend dependency), and P4 (the designer witnesses autonomy clearing real work, gated, on a budget). It is also the **first real test of `vend steer`/`survey` on a thin domain seed** (they were authored against the vend repo) — the make-or-break A3 risk, now de-risked by the shipped steer self-referential-demotion fix (E-044).

## Done looks like

A designer clones `examples/templates/hackathon-seed/` (an Astro + React app wired to auto-deploy to Cloudflare), runs `vend init --template hackathon` + `vend doctor` (green), edits a one-line `SEED.md`, and runs `vend steer` then `vend work` — clearing ≥1 real slice. While the app previews in their browser, the **SVG work-graph board renders beside it** (status columns, blocked edges in red), so they understand what's happening without reading code. The drive is captured against a **gold-mastered `EXPECTED-OUTCOME.md` produced from a real live run** (the closing ticket), and the cleared run accrues a genuine forward-E1 record (the Frontier-1 second payoff).
