Choosing (C) recommits to a proof the field just weakened; (A)/(B) commit to the allocation thesis the last three epics have actually been building.
- **Options:**
  1. Pull OVERFLOW routing next — the ratified ladder's literal next rung and the direct continuation of the arc just closed.
  2. Jump to the CROSS-VENDOR review gate first — skip overflow for the highest-differentiation rung ('no single lab can ship').
  3. Return to Frontier-1: fund a genuine-intervention forward-E1 sweep to move the autonomy keystone off 0/10.
- **Vend recommends:** Overflow (A). It's the ratified ladder's next rung, the honest continuation of the just-closed routing arc, and matches the 0/10 autonomy honesty rather than fighting it — hold cross-review as the immediate follow-on.

### Fork — On the visual surface (Frontier 4): keep building SVG polish, or validate hypothesis 2b (non-devs actually want a visual surface) before spending more on it?
- **Why it matters:** Hypothesis 2b (non-devs want a VISUAL surface) is still hypothesis-grade, so option (C) risks overproducing a surface no one pulls — the exact waste the board exists to prevent. But (A) alone is blocked on a real designer's time, and a jargon-leaking face would bias the probe toward a false negative ('just a reframed text wall').
- **Options:**
  1. Run the render-and-watch probe FIRST — hand a real designer one rendered .svg and measure where she gets lost, before any more surface work.
  2. Ship the card-face jargon strip first (the #1 named weak link), THEN run the probe on the improved surface.
  3. Build all remaining SVG polish (jargon + color-blind + layout stability), defer the probe.
- **Vend recommends:** Ship the jargon strip, then probe (B). It's the one cheap fix most likely to change the probe's outcome, and it's a brand-voice mandate regardless of what 2b returns — so it's never wasted.
