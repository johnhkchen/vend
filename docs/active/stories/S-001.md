Part of **epic E-001 (dispense-slice)**. The foundation of Vend's single metered
lever: a Bun/TypeScript project with a `bun run check:*` surface, the `claude -p`
dispense seam, a budget control, and a countable run log. No play yet — this is
the infrastructure the `DecomposeEpic` play (S-002) composes.

**Value:** proves the stack carries P1/P3/P7 and lays the scaffold + check surface
that E-002 (CI) and every later epic depend on.

## Tracks

- **T-001-01** (scaffold) gates everything.
- After it, **T-001-02** (seam), **T-001-03** (budget), **T-001-04** (log) run in
  **parallel** — separate modules, no shared files (max_threads caps concurrency).

## DAG

```
           ┌─ T-001-02  executor seam (claude -p)
T-001-01 ──┼─ T-001-03  budget control
 scaffold  └─ T-001-04  countable run log
```
