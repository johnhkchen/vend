Part of **epic E-046 (typed-dag-fan-out-join-substrate)** — the foundational slice of the v1 vision's
"typed, graph-structured agent orchestration." Today the engine casts single plays plus **one linear
propose→decompose chain** (`chain-core.ts` `runChain` / `chain.ts` `castChain`); the graph the vision
promises doesn't exist. This story generalizes that chain — a *path* graph — into a typed DAG with the
first genuinely non-linear shapes: **fan-out** (one node → several) and **join** (several → one).

The decomposition mirrors the chain's **pure-core ⊥ impure-shell** split (so the pure core stays
addon-free + fake-testable, the `chain-core.test.ts` discipline), and respects E-046's scope: build the
typed graph model + a topological executor proving fan-out + join; **out of scope** (follow-on epics) —
conditional edges, cycle/error *semantics*, cross-branch budget accounting, the open-model runner.

Model → executor → shell+example: T-046-01 (the typed node/edge model + a pure topological sort;
acyclicity validated) → T-046-02 (`runGraph` pure core — multi-upstream join threading, fan-out, halt
the dependent subgraph; proven with fakes) → T-046-03 (the `castGraph` impure shell with parallel
independent branches + a deterministic worked example (fan-out + join over stub nodes) + the AC test
that fails against the old linear-chain engine). Epic-scoped ids (R6 — `046` clean).

## DAG

```
T-046-01  typed graph model (DagNode/DagEdge/DagSpec) + pure topoSort (acyclic validated) — fully unit-tested
   └─ T-046-02  runGraph pure core — generalize runChain: join threads many upstreams · fan-out · halt the dependent subgraph (fakes)
        └─ T-046-03  castGraph shell (adapt→castPlay, parallel branches) + worked example (fan-out+join, stub nodes) + test that fails vs the old linear engine
```

**Out of this slice:** conditional edges; cycle/error *semantics* (cycles are detected + refused, not
run); budget accounting across parallel branches; a LIVE real-play graph cast (deterministic stub-node
example here; a metered real cast is downstream); the open-model runner (the P6 consumer).
