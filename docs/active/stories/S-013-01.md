measured budget envelope per play, so the Confirm default (IA-6) is earned, not
guessed. The first rung of the "work for 2 hours" macro mechanic. Design settled in
`information-architecture.md` IA-12…IA-15.

Shared contract + foundation first (R12/R3 — the run-record shape + a pure log
reader), then the pure recalibration core + the measured default surface (R1).
Epic-scoped ids (R6 — `013` is clean).

