the reviewer seat: whenever a complement reviewer WOULD resolve under the current config

and goes red with a named check + fix-it hint when it can't. With default (unprovisioned) config

review") rather than staying silent.

with a provisioned-but-unreachable reviewer goes red naming the reviewer seat and the fix; the
probe reuses S-076-01's resolution and T-074-01-01's dispensability probe — no new probe
mechanism.

**Honest boundary:** FREE — probe-level, no metered cast. Green must mean "everything this cast
will dial is reachable"; silence about a configured reviewer is the lie this story removes.
