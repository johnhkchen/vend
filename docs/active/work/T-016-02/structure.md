## `src/play/expand-effect.ts` (new) — the addon-free effect

Imports: `mkdir, writeFile` (`node:fs/promises`); `join` (`node:path`); **type-only** `Signal`
(`baml_client/index.ts`), `CastContext`/`EffectResult` (`engine/play.ts`); value `renderSignalRow`
(`./expand-core.ts` — pure, addon-free). **No BAML, no spawn** → testable against a temp dir.

```
export const STAGING_DIR = "docs/active/pm/staged";   // the PM-desk inbox (D2)

export interface ExpandFragmentInputs {               // what render consumes + gate/effect ctx
  readonly fragment: string;
  readonly charter: string;                           // value-link greps it for live P#/N#
  readonly project: string;                           // go-and-see snapshot
}

export function slugify(what: string): string;        // pure; lowercase, non-alnum→-, fallback "signal" (D3)

export function renderStagedSignal(signal: Signal): string;  // pure; heading + demand row + pull string + trailer (D4)

export async function expandFragmentEffect(           // the one IMPURE verb
  signal: Signal,
  ctx: CastContext<ExpandFragmentInputs>,
): Promise<EffectResult>;
```

`renderStagedSignal(signal)` body (pure, deterministic):
- `# ${signal.what}` heading
- the `demand.md` table header (`| Signal | Value | Budget (envelope) | Status |` + the
  `|---|---|---|---|` rule) followed by `renderSignalRow(signal)`
- a `## Pull this` block with a fenced `vend chain "<what> — <why>"` line (the signal string)
- `_Staged by Vend's \`expand-fragment\` play — not promoted; pull to clear._`

`expandFragmentEffect`:
- `dir = join(ctx.projectRoot, STAGING_DIR)`
- `path = join(dir, \`${slugify(signal.what)}.md\`)`
- `mkdir(dir, {recursive:true})`; `writeFile(path, renderStagedSignal(signal), "utf8")`
- returns `{ ok: true, detail: \`staged ${path}\`, artifacts: [path], produced: path }`
- a genuine fs failure THROWS (uncontracted — the `captureNoteEffect` rule); no `outcome` relabel
  (no id collision possible — overwrite-by-slug is intended).

Header doc-comment records: addon-free + impure; the no-id / overwrite-by-slug divergence from
`proposeEpicEffect`; writes ONLY under `docs/active/pm/` (never the board).

## `src/play/expand-fragment.ts` (new) — the shell

Imports: `readFile` (`node:fs/promises`); `join` (`node:path`); value `b`
(`baml_client/sync_client.ts`); **type-only** `Signal` (`baml_client/index.ts`); `extractPromptText`
(`../baml/decompose-bridge.ts`); `registry`, type `Card`/`Play` (`../engine/play.ts`); `castPlay`

`type ExpandFragmentInputs` (`./expand-effect.ts`).

```
export const PLAY = "expand-fragment";
export type { RunSummary } from "../engine/cast.ts";   // re-export (type-only), the propose idiom

const EMPTY_SIGNAL: Signal = { what:"", why:"", tier:"Keystone" as Signal["tier"],
                               budget:"", advances:[], grounding:"", readiness:"" };  // D6

export function parseExpandFragment(text: string): Signal;   // try b.parse … catch → EMPTY_SIGNAL

export const expandFragmentPlay: Play<ExpandFragmentInputs, Signal> = {
  name: PLAY,
  render: (i) => extractPromptText(b.request.ExpandFragment(i.fragment, i.charter, i.project) as …),
  parse: parseExpandFragment,
  gates: (signal, ctx) => clear(signal, { charter: ctx.inputs.charter }),   // ExpandClearContext
  effect: expandFragmentEffect,
  budget: { timeMs: 1_200_000, tokens: 12_000 },                            // D7
  card: { color: ["blue","green"], type: "permanent", rarity: "rare" } satisfies Card,
};
registry.register(expandFragmentPlay);                  // self-register at module load

export interface ExpandFragmentOptions {
  readonly fragment: string;                            // the rough fragment — the cast subject
  readonly budget: Budget;
  readonly projectRoot?: string;
  readonly model?: string;
  readonly runId?: string;
  readonly transcriptDir?: string;
}

export async function assembleExpandFragmentInputs(opts): Promise<ExpandFragmentInputs>;  // IMPURE
export async function castExpandFragment(opts: ExpandFragmentOptions): Promise<RunSummary>;
```

`assembleExpandFragmentInputs`: `root = opts.projectRoot ?? cwd`; read `CHARTER_PATH`; list
`stories`/`tickets` ids; `project = buildProjectSnapshot({root, srcFiles:[], stories, tickets})`;
return `{ fragment: opts.fragment, charter, project }`. No `existingEpicIds` (a signal mints no id).

`castExpandFragment`: `inputs = await assemble(opts)`; `return castPlay(expandFragmentPlay, inputs,
opts.budget, { subject: opts.fragment, projectRoot: root, model, runId, transcriptDir })`.

Header doc-comment: the IMPURE shell (value-imports `b`); NO bun-test value-imports it (its logic
is the engine core + expand-core.test + expand-effect.test + the expand.test bridge); the
fourth registry entry — `fragment → signal`, one notch upstream of propose's `signal → epic`.

`tier:"Keystone"`). `seedRoot()` → `mkdtemp`. Blocks:

  and the `expand-fragment` trailer; **assert no `demand.md` / `epic/` write** (the dir doesn't

  `gate-failed`, `materialize:false`, `gate === "honest-empty"`; an ungrounded signal (blank
  `grounding`) → `gate-failed`, `gate === "read-never-invent"`.
- **slugify** — `"Register expandFragmentPlay"` → `"register-expandfragmentplay"`; all-punct →
  `"signal"`.
